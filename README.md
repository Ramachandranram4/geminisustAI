# SmartCity Sentinel | AI Emergency Response

SmartCity Sentinel is a professional emergency intelligence platform that uses Genkit and Gemini to analyze media feeds (images/video) and provide real-time situational awareness for smart cities.

## Features

- **Instant Incident Detection**: Identifies Fire, Floods, Accidents, and more from uploaded media.
- **Real-time GIS Mapping**: Integrated OpenStreetMap (Leaflet) showing incident locations and nearby resources.
- **Multilingual Support**: AI-driven regional language translation for emergency instructions.
- **Emergency Agent**: Conversational AI to guide users through immediate safety precautions.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **AI**: Genkit with Google Gemini 2.5 Flash
- **Maps**: OpenStreetMap with React-Leaflet
- **Styling**: Tailwind CSS & ShadCN UI
- **Language**: TypeScript
